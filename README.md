# Blacksmith

Claude Code를 위한 스킬 대장간.

## Installation

```bash
/plugin marketplace add eatnug/blacksmith
```

---

## Catalog

### clarify

> 요구사항이 애매할 때, 먼저 물어보고 시작합니다.

```bash
/plugin install clarify@blacksmith
```

**When to use**
- 요구사항이 모호하거나 불완전할 때
- 여러 해석이 가능할 때
- 잘못 이해하고 삽질하기 전에

---

### split

> 복잡한 작업을 작은 단위로 쪼갭니다.

```bash
/plugin install split@blacksmith
```

**When to use**
- 여러 파일/컴포넌트에 걸친 작업
- 한 번에 하면 망할 것 같은 리팩토링
- 진행 상황을 추적하고 싶을 때

---

### forge

> 반복되는 패턴을 발견하고 새 스킬로 만듭니다.

```bash
/plugin install forge@blacksmith
```

**When to use**
- 같은 요청을 여러 번 하고 있을 때
- "이거 자동화하면 좋겠다" 싶을 때
- 팀 워크플로우를 스킬로 만들고 싶을 때

---

### look-back

> 작업 후 실수를 찾아내고 수정합니다.

```bash
/plugin install look-back@blacksmith
```

**When to use**
- 뭔가 찜찜할 때
- 작업 완료 전 최종 점검
- "진짜 다 한 거 맞아?" 확인용

---

### format-response

> 응답 포맷을 커스터마이징합니다.

```bash
/plugin install format-response@blacksmith
```

**When to use**
- 터미널 크기에 맞는 출력이 필요할 때
- 응답 형식을 통일하고 싶을 때
- template.md로 나만의 포맷 정의하고 싶을 때

---

## Full Set

전부 설치:

```bash
/plugin install clarify@blacksmith
/plugin install split@blacksmith
/plugin install forge@blacksmith
/plugin install look-back@blacksmith
```
