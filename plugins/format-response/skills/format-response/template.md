# Response Template

## 기본 구조

응답은 다음 구조를 따른다:

1. **핵심 답변** - 질문에 대한 직접적인 답 (1-3줄)
2. **상세 내용** - 필요한 경우에만
3. **다음 단계** - 액션이 필요한 경우에만

## 포맷팅 규칙

### 테이블
- 터미널 너비에 맞게 조정
- 120+ cols: 전체 표시
- 80-119 cols: 축약
- <80 cols: 리스트로 전환

### 코드
- 한 줄이 너비 초과시 적절히 줄바꿈
- 들여쓰기 유지

### 경로
- 길면 `...`로 축약
- 핵심 부분은 유지

### 목록
- 5개 이하: inline 가능
- 6개 이상: vertical
